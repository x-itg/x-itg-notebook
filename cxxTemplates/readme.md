# C++ STL代码示例

1. 安装
   
   ```
   sudo apt-get install build-essential gcc g++ gdb cmake libboost-all-dev
   ```
   
   ```
   在VSCode上安装插件
   Remote - WSL
   点击左下方的连接远程系统按钮，连接WSL，然后在打开后的VSCode上安装C++插件:
   C/C++
   C++ Intellisense
   CMake
   Cmake Tools
   ```

2. 创建build文件夹

3. 进入build文件夹

4. cmake ..

5. make
